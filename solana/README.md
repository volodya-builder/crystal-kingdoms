# $CRYSTAL — тестовый токен (Solana Devnet) + мост Ruby↔$CRYSTAL

Безопасный прототип на **devnet** (тестовая сеть, реальных денег нет).

## 0. Подготовка (один раз)
```bash
cd solana
npm install
```

## 1. Создать токен $CRYSTAL
Сначала скопируй адрес своего кошелька Phantom (переключи Phantom на **Devnet** в настройках!), затем:
```bash
node create-token.js <АДРЕС_ТВОЕГО_КОШЕЛЬКА>
```
- создастся монета `$CRYSTAL`, казна (`treasury.json` — **СЕКРЕТ, не публикуй и не коммить!**),
- эмиссия 100 000 000 → казна, и тебе начислят 5000 $CRYSTAL для теста.
- в конце выведет **MINT** и **TREASURY** — они уже записаны в `config.json`.

> Если airdrop SOL не прошёл — пополни казну вручную на https://faucet.solana.com (адрес из вывода).

## 2. Запустить мост-сервер
```bash
node bridge-server.js
```
Сервер на `http://localhost:8787` — он подписывает операции ключом казны (минт игроку, проверка депозита).

## 3. В игре
- Кнопка **💎 Кошелёк** (рядом с ⚙) → «Подключить Phantom».
- Видишь баланс $CRYSTAL.
- **Ruby → $CRYSTAL** (вывод): игра спишет Ruby и сервер начислит токены на кошелёк.
- **$CRYSTAL → Ruby** (ввод): Phantom подпишет перевод токенов в казну, сервер подтвердит — игра начислит Ruby.

Курс по умолчанию **1 Ruby = 1 $CRYSTAL** (меняется в `bridge-server.js`, `RATE`).

## ⚠️ Это девнет-прототип
- Сервер доверяет клиенту (для теста). Для продакшна: проверять Ruby на бэке, лимиты, антифрод, и перенести сервер с локалки на хостинг/Edge Function, ключ казны — в секреты.
- Реальный mainnet-запуск токена = юрист + аудит. Сначала отлаживаем экономику на devnet.

## Файлы
- `create-token.js` — создаёт токен и казну.
- `bridge-server.js` — мост (минт/проверка переводов).
- `config.json` — MINT/TREASURY (создаётся скриптом).
- `treasury.json` — **СЕКРЕТНЫЙ** ключ казны (в .gitignore!).
